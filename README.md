# cloud_radar


## setting up python

### Create venv

```bash
python3 -m venv venv
```

### Activate venv

- Linux

```bash
source venv/bin/activate
```

- windows, command promt

``` shell
venv\Scripts\activate
```

- windows, powershell

``` powershell
.\venv\Scripts\Activate.ps1
```

### Install requirements

```bash
pip install -r requirements.txt
```
